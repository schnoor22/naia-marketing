# NAIA Marketing Website

Coming soon landing page for NAIA - Industrial AI Framework.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Deployment

Deploy to Vercel by connecting this repo to your Vercel account.

## Setup Instructions

1. Create a new GitHub repo: `naia-marketing`
2. Push this project to that repo
3. Connect to Vercel
4. Point your domain nameservers to Vercel
5. Done!
